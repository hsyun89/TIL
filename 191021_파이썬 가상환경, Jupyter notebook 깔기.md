# 파이썬 가상환경, Jupyter notebook 깔기

1. git bash 에서 가상환경 만들기

   ```bash
   $python -m venv venv
   ```

2. ```bash
   $source ./venv/Scripts/activate
   ```

3. ```bash
   $python -m pip install --upgrade pip
   ```

4. ```bash
   $pip install jupyter notebook
   ```

5. ```bash
   $jupyter notebook
   ```
